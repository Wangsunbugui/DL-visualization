项目名称：深度学习模型训练可视化平台

——————————————————————————————————————————————

项目概述:
本项目的目的是构建一个深度学习模型训练可视化平台。平台的前端提供模型和参数的选择,
用户选择后会对应启动模型训练,训练过程会产生特征图,这些特征图会被发送到前端,
前端会实时显示这些特征图,让用户从特征图变化体会到模型训练的过程。

——————————————————————————————————————————————

安装和使用：

安装Python 3.10和相关依赖包

使用命令安装依赖包:

pip install -r requirements.txt

启动服务端:

python app.py


访问 http://localhost:5002 使用平台

功能特性
模型选择:提供多种深度学习模型供用户选择
参数调整:用户可以调整模型的训练参数,如学习率、优化器等
实时显示特征图:训练过程中的特征图实时发送到前端显示,让用户直观感受模型的变化
